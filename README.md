# Indian Knowledge System (IKS) Documents

This repository contains interactive study guides and documentation for the Indian Knowledge System (IKS) - AMC 1001.

## Contents
- `iks.html`: An interactive, premium web guide covering Chapters 1-6 of the IKS curriculum.

## Features
- Interactive 6-chapter breakdown.
- Searchable topics and concepts.
- Quick Knowledge Check (Quiz).
- Cross-chapter connection maps.
- Premium, responsive design with liquid-glass aesthetics.
